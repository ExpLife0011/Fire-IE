Fire-IE by yxl

Switch to IE engine in one click.

A Firefox add-on let you switch to IE engine in one click. It's time to give up your Internet Explorer.

Install

Fire-IE is a standard Firefox addon. Download the addon file fireie-0.4.5-unified.xpi and open with Firefox to install it.
